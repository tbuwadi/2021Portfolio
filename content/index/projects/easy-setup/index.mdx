A work from home companion that increases the wellbeing of those who have #zoomfatigue caused by the overload of video call meetings. Using python, pyvirtualcam and OpenCV, we integrated a virtual camera that can run on any video calling platform. We used multithreading in combination with OpenCV to process video frame data, and allow asynchronous control of the virtual camera while providing a user interface at the same time.
